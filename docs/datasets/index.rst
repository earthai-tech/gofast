*********
Datasets
*********

.. currentmodule:: gofast.datasets

The :mod:`gofast.datasets` module is dedicated to providing a variety of datasets and tools for data loading and simulation. These 
datasets span multiple domains, including biology, finance, healthcare, and more. They are designed to facilitate machine learning 
tasks such as classification, regression, and clustering, as well as to support data analysis and research.
